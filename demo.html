轻弹窗

演示1.只有一个确认
代码
        box.alert("我出来了,点击确定弹出alert",function(d){
            if(d){
                alert("弹出了一个alert");
            }
        },{
            title:'自定义的提示标题'
        })
演示2.有确认和取消
代码
        box.confirm("确定和取消弹出的提示不一样哦",function(d){
            if(d){
                alert("yes");
            } else {
                alert("no")
            }
        },{
            title:'自定义的提示标题'
        })
